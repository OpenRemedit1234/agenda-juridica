# ⚖️ Agenda Jurídica

Aplicación web sencilla desarrollada con Flask para la gestión de citas y notas en un servicio jurídico.

## 📌 Descripción

Esta aplicación permite organizar de forma rápida y clara la información diaria de un despacho jurídico, incluyendo citas con clientes y notas relacionadas con casos.

Está pensada para ser fácil de usar y ligera, sin necesidad de bases de datos complejas.

---

## 🚀 Funcionalidades

* 🔐 Sistema de login
* 📅 Crear citas (con fecha y hora)
* 📝 Crear notas
* ❌ Eliminar citas y notas
* 📋 Visualización organizada de la información

---

## 🛠️ Tecnologías utilizadas

* Python
* Flask
* HTML + CSS
* JSON (almacenamiento de datos)

---

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio:

```
git clone https://github.com/TU_USUARIO/agenda-juridica.git
```

2. Entrar en la carpeta:

```
cd agenda-juridica
```

3. Instalar dependencias:

```
pip install flask
```

4. Ejecutar la aplicación:

```
python app.py
```

5. Abrir en el navegador:

```
http://127.0.0.1:5000
```
